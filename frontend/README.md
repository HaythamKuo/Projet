# Doll Soldier - Frontend

> **電商專案---前端架構，由 React 與 Vite 建構**

## 技術棧

React, Vite, React Router Dom, Redux-toolkit, Redux-toolkit Query, Styled-Components, Framer Motion, Vitest, MSW, Testing-Library

## 運行環境

```

Node 版本 >= 20

React 版本 >= 18.3

Vite 版本 >=6.2

```

## 主要功能

- 交易功能
- 會員登入/註冊驗證
- 會員狀態管理(是買家也是賣家)
- 商品搜尋功能
- 綠界科技交易
- UI 樂觀更新
- 單位測試

## 由此開始

1. 下載依賴項:

```bash

npm install

```

2. 創建一個 `.env` 載入環境變數，如後端開發伺服器

3. 運行前端環境:

```bash

npm run dev

```

4. 預設路由為[http://localhost:5173/](http://localhost:5173/)
